How it works

Up and running in three steps

1

Clone the repo

Fork or clone to get all 32 skills and the install scripts. Everything is plain Markdown — no dependencies.

terminal
git clone https://github.com/ majidraza1228/claude-skills.git
2

Install for your tool

One command installs all skills globally for Claude Code, or exports them formatted for Cursor, VS Code, Codex, and more.

install
# Claude Code ./scripts/install-skills.sh --global # Any other tool ./scripts/export-skills.sh cursor
3

Trigger with natural language

In Claude Code, skills auto-route from what you type. In other tools, @mention or use #file: to load the right skill.

Claude Code
"review my code" → /code-review "plan this feature" → /planning "write a ticket" → /ticket-writer
